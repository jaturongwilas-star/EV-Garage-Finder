เข้าสู่ระบบแอดมิน

การเพิ่ม/แก้ไขฐานข้อมูลอู่สงวนไว้สำหรับแอดมินเท่านั้น กรุณากรอกชื่อผู้ใช้และรหัสผ่านเพื่อดำเนินการต่อ

อัปโหลดข้อมูลอู่จาก Excel

อัปโหลดไฟล์ .xlsx / .xls / .csv ตามรูปแบบ Template — ระบบจะอ่านหัวคอลัมน์ภาษาไทยอัตโนมัติ (ประเภทรถ EV และ งานที่รับซ่อม คั่นหลายค่าด้วยจุลภาค) แล้วบันทึกขึ้น Google Sheet โดยตรง ให้ทุกคนเห็นข้อมูลเดียวกัน

Key นี้ถูกเก็บไว้ในหน่วยความจำชั่วคราวของเบราว์เซอร์ (sessionStorage) เท่านั้น — หายไปเองเมื่อปิดแท็บหรือกด "ออกจากระบบแอดมิน" ไม่ถูกบันทึกลงไฟล์ใดๆ ทั้งสิ้น
Key นี้ตั้งค่าไว้ที่ Google Apps Script (Script Properties > API_KEY) ไม่ได้อยู่ในโค้ดเว็บนี้เลย ดูวิธีตั้งค่าได้ที่ไฟล์ apps-script/Code.gs หรือ README
